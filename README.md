React Native (expo)

# 실행

vsCode에서 코드를 작성하고 expo go를 활용하여 화면을 구성하였습니다.

# 실행 결과

초기 UI 구성: 중첩 네비게이터를 사용하여 탭 네비게이터와 스택 네비게이터를 동시에 사용했습니다.
             만보기 기능을 구현하기 위해 프로그레스 바를 구현하여 넣었습니다.
